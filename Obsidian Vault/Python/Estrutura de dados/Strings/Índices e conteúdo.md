Índice é o número para representar a posição de um conteúdo (caractere). Isso quer dizer que o primeiro caractere de uma string é de posição ou índice 0. 
Exemplo:
A L F A C E
0 1  2  3 4 5

Para acessar os caracteres de uma string, devemos informar o índice ou posição do caractere entre colchetes.
Exemplo:
![[Pasted image 20260319144558.png]]

Índices negativos - um índice negativo começa a contar do último elemento. Assim, -1 representa o último elemento, -2 o penúltimo e assim por diante.
