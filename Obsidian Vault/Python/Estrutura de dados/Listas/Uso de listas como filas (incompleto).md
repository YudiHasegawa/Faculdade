Uma lista pode ser utilizada como fila se obedecer a  certas regras de inclusão e eliminação. Em uma fila, a inclusão é sempre realizada no fim, e as remoções são feitas no início. Se diz que o primeiro a chegar é o primeiro a sair (**FIFO - First In First Out**). 

Observação: Para usar uma lista como fila, provavelmente irá utilizar os seguintes métodos: `list`,
`range`, `len`, `.pop()`, `.append()`, `input`.

Exemplo:
`adicionar Programa 6.7 da página 107!!!`
