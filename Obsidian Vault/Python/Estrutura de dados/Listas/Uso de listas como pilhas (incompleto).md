Para usar listas como pilhas, é necessário obedecer uma regra de pilha: novos elementos são adicionados ao topo; a retirada de elementos também é feita pelo topo.

Observação: Para usar uma lista como pilha, provavelmente irá utilizar os seguintes métodos: `list`,`range`, `len`, `.pop()`, `.append()`, `input`.

Exemplo:
`adicionar Programa 6.8 da página 108!!!`
