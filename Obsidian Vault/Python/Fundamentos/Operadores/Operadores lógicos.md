Utilizado para agrupar operações com lógica booleana.

Operador not -  realiza uma operação de negação/inversão.
Exemplo:
V = True
not V1 = False

Operador and - resulta em verdadeiro apenas quando seus dois operadores forem verdadeiros.
Exemplo:
V1 = True
V2 = True
V1 and V2 = True

V1 = True
V2 = False
V1 and V2 = False

V1 = False
V2 = False
V1 and V2 = False

Operador or - apenas um dos seus dois operadores precisa ser verdadeiro para resultar em verdadeiro.
Exemplo:
V1 = True
V2 = True
V1 or V2 = True

V1 = True
V2 = False
V1 or V2 = True

V1 = False
V2 = False
V1 and V2 = False
