É a combinação de operadores lógicos.

Ordem de precedência das operações:
1. not
2. and
3. or

Exemplo:
True or False and not True
True or False and False
True or False
True

Os operadores relacionais também podem ser utilizados em expressões com operadores lógicos.
Exemplo (salário > 1000 and idade > 18):
salário = 100
idade = 20
100 > 1000 and 20 > 18
False and True
False

salário = 2000
idade = 30
2000 > 1000 and 30 > 18
True and True
True
